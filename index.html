Lexical Analyzer

A simple lexical analyzer that tokenizes your code input

Code Input

How to use:

  • Enter your code in the editor above
  • Click "Analyze Code" to tokenize your code
  • Tokens will be displayed in the results table with color coding
  • Supported token types: Keywords, Identifiers, Operators, Numbers, Strings, Comments, Punctuation

Examples of tokens:

Keywords: int, if, return
Identifiers: x, message, main
Operators: =, +, >, ==
Numbers: 10, 3.14, 0
Strings: "Hello", "World"
Comments: // This is a comment
